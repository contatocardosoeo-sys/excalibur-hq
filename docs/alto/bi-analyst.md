# BI Analyst — Excalibur HQ

## KPIs por setor

### CEO
- Receita do mês (total e projetada)
- MRR + crescimento %
- LTV / CAC / Payback
- Churn rate
- Margem líquida
- Caixa atual

### SDR (Trindade)
- Leads recebidos / contatos / agendamentos / comparecimentos / vendas
- Valor das vendas (R$)
- Taxa de conversão por etapa
- Atingimento de meta mensal

### Comercial (Guilherme - Closer)
- Reuniões realizadas
- Propostas enviadas
- Fechamentos
- MRR gerado
- Comissão acumulada

### Tráfego (Guilherme - CMO)
- Investimento total
- Leads gerados
- CPL real vs baseline
- Funil completo: leads → agend → comp → qualif → fech → fat
- Gargalo identificado

### CS (Medina)
- Clínicas ativas
- Score médio (saúde)
- Em risco (score < 60)
- Tarefas pendentes
- Jornada D0-D90 por cliente
- Inadimplentes

### Financeiro
- A receber / A pagar
- Caixa do mês
- Tx pagamento %
- Inadimplência
- Top despesas por categoria

## Princípios de visualização

### Cores semânticas
- Verde = bom (acima de 80% meta)
- Amarelo = atenção (50-80%)
- Vermelho = crítico (< 50%)

### Hierarquia
1. KPI principal grande no topo
2. Metas com barra de progresso
3. Detalhes em tabela ou grid
4. Inadimplentes/alertas em destaque

### Não fazer
- Gráficos desnecessários (números são mais claros)
- Cores arco-íris
- Visualizações que demandam interpretação
- Múltiplos eixos em um gráfico

## Auto-refresh
- Páginas críticas: 60s
- Dashboards executivos: 120s
- Relatórios estáticos: manual
