# ⚔️ EXCALIBUR HQ — Backoffice Interno

## IDENTIDADE
Este é o **excalibur-hq** — backoffice da empresa Excalibur.
Só a equipe Excalibur acessa. Foco: gestão de clientes, CS, comercial.

## STACK
Next.js 16 + TypeScript + Tailwind CSS 4 + Supabase + Vercel
URL: excalibur-hq.vercel.app → hq.excalibur.com.br

## MÓDULOS
- Dashboard executivo (MRR, churn, clientes)
- Gestão de clientes (clínicas)
- Jornada (onboarding D0-D90)
- Health Score
- Comercial (CAC, LTV, pipeline)
- Marketing
- Treinamento
- Customer Success
